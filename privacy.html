Política de Privacidade — B3 Quick Quote

Última atualização: 25/12/2025

Resumo: A extensão B3 Quick Quote não coleta, não vende e não compartilha dados pessoais. Ela apenas consulta cotações públicas de mercado quando você solicita.

Dados coletados

Nenhum dado pessoal é coletado.

Dados armazenados no seu dispositivo

  • Opcionalmente, a extensão pode armazenar localmente o último ticker digitado para facilitar o uso (via chrome.storage).

Esse dado fica somente no seu navegador e não é enviado para nenhum servidor do desenvolvedor.

Fontes de dados de mercado

Quando você busca um ticker ou carrega as “maiores altas/baixas”, a extensão faz requisições HTTPS para endpoints do Yahoo Finance (ex.: query1.finance.yahoo.com) para obter informações públicas de mercado.

Compartilhamento de dados

A extensão não compartilha dados com terceiros. As requisições ao Yahoo Finance contêm apenas os símbolos consultados (tickers) necessários para retornar as cotações.

Permissões

  • storage: para salvar localmente o último ticker digitado (opcional).
  • host permissions (Yahoo Finance): para permitir a consulta de cotações.

Publicidade, analytics e rastreamento

A extensão não exibe anúncios, não usa analytics e não faz rastreamento de navegação.

Contato

Se você precisar de suporte ou desejar remover dados locais, basta desinstalar a extensão (isso remove também o armazenamento local).
