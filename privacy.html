Política de Privacidade — B3 Quick Quote

Última atualização: 25 de dezembro de 2025

Resumo: A extensão B3 Quick Quote não coleta, vende ou compartilha dados pessoais. Ela apenas consulta cotações públicas de mercado quando você solicita.

Dados Coletados

Nenhum dado pessoal é coletado.

Dados Armazenados no Seu Dispositivo

  • Opcionalmente, a extensão pode armazenar localmente o último ticker digitado para facilitar o uso (via chrome.storage).
  • Cotações recentes são armazenadas em cache para exibição quando o mercado estiver fechado.

Esses dados ficam apenas no seu navegador e não são enviados para nenhum servidor do desenvolvedor.

Fontes de Dados de Mercado

Quando você busca um ticker ou carrega as "maiores altas/baixas", a extensão faz requisições HTTPS para a API do brapi.dev para obter informações públicas do mercado brasileiro (B3).

Compartilhamento de Dados

A extensão não compartilha dados com terceiros. As requisições ao brapi.dev contêm apenas os símbolos consultados (tickers) necessários para retornar as cotações.

Permissões

  • storage: para salvar localmente o último ticker digitado e cache de cotações.
  • host permissions (brapi.dev): para permitir a consulta de cotações.

Publicidade, Analytics e Rastreamento

A extensão não exibe anúncios, não usa analytics e não faz rastreamento de navegação.

Contato

Se você precisar de suporte ou desejar remover dados locais, basta desinstalar a extensão (isso remove também o armazenamento local).
